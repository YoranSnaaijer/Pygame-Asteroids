# Pygame-Asteroids

A simple asteroids game built with Pygame.

## Usage

Run the game with default QWERTY controls (WASD + Space):

```bash
python main.py
```

To use Colemak controls (WARS + Space), add the `--colemak` flag:

```bash
python main.py --colemak
```

### Controls

- **QWERTY (default)**: W (forward), A (rotate left), S (backward), D (rotate right), Space (shoot)
- **Colemak**: W (forward), A (rotate left), R (backward), S (rotate right), Space (shoot)
